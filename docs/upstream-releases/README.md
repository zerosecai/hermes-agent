# Upstream Hermes Agent release notes

This directory holds the historical release notes for **Hermes Agent v0.2.0 through v0.12.0** as written by Nous Research. They were inherited from upstream and moved here from the repo root in Kuma's Batch 7 cleanup so the root tree isn't dominated by ~360 KB of pre-fork history.

The content is preserved verbatim. **Don't edit these files** — rewriting upstream's own release narrative would be revisionist. If you find a factual error, file it upstream at [NousResearch/hermes-agent](https://github.com/NousResearch/hermes-agent).

For Kuma-specific release notes (everything from `v0.12.0-kuma1` onwards), see the project root [`CHANGELOG.md`](../../CHANGELOG.md).

For ongoing upstream releases past `v0.12.0`, see the [upstream releases page](https://github.com/NousResearch/hermes-agent/releases).
